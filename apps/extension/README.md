<h1 align="center">GitStudio</h1>

<p align="center">
  <b>A free, open-source, JetBrains-grade Git experience for VS Code & Cursor — the whole workflow, not just one piece.</b>
</p>

---

GitStudio brings the parts people love from GitLens, GitKraken, and JetBrains IDEs into **one unified, free, open-source** extension:

- **Commit graph** — a real, virtualized branch/commit graph (fast at 10k–100k commits).
- **Blame & authorship** — inline current-line blame, full-file annotations, a code-age heatmap, and rich command hovers.
- **History & timeline** — file history, line history (blame-over-time), revision navigation, and a reflog time-machine.
- **Diff & 3-pane merge** — side-by-side and unified diffs with word-level highlighting, plus a JetBrains-style three-pane merge editor.
- **Staging & commit** — hunk- and line-level staging, amend, sign-off, and a guided commit box.
- **Interactive rebase** — a drag-to-reorder rebase editor wrapped in a universal, reflog-powered **Undo** safety net.
- **Branches, remotes, stashes, tags, worktrees** — rich sidebar views and operations.
- **GitBrain (optional AI)** — bring-your-own-key commit messages, explain-this-diff, and more. Off by default, never required.

Free on private repos, no account, telemetry off by default. Built on stable VS Code APIs so it runs identically on VS Code and Cursor (Open VSX).
